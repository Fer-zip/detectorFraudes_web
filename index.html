¿Qué es PredicFraud?

Esta aplicación web está diseñada para ayudarte a detectar posibles estafas digitales. Utilizando un modelo de inteligencia artificial entrenado con miles de ejemplos reales, analizamos capturas de pantalla de correos, mensajes o chats sospechosos y te mostramos qué tan probable es que se trate de una estafa.

El objetivo es brindarte una herramienta rápida y accesible para protegerte de fraudes, sin necesidad de conocimientos técnicos. Solo sube una imagen y deja que la IA haga el resto.

 Ver el repositorio

¿Como usar PredicFraud?

Usar nuestra aplicación es muy sencillo. Solo sigue estos pasos:
  • 1. Toma una captura de pantalla del correo electrónico, mensaje o conversación que crees que podría ser una estafa.
  • 2. Cárgala en la aplicación desde tu dispositivo.
  • 3. En segundos, recibirás un porcentaje de probabilidad que indica si el contenido es fraudulento.
¡Así de fácil! No necesitas registrarte ni dar datos personales. Tu seguridad es lo primero.

Preguntas frecuentes

¿Cómo usar el detector de estafas?

Para usar el detector de estafas se tiene que cargar una captura de pantalla del correo o mensaje sospechoso, o una imagen. El sistema analizará la imagen y te dirá si podría tratarse de una estafa, usando un modelo entrenado con Teachable Machine.

¿El servicio es gratuito?

Sí, este servicio es completamente gratuito. Fue creado con fines educativos y de concientización sobre los riesgos en línea.

¿Es seguro?

Sí. El sistema no guarda ni comparte las imágenes cargadas. Todo el análisis se realiza localmente en tu navegador usando modelos de aprendizaje automático. Tu seguridad es primero.
